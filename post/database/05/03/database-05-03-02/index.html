5.3.2 最小函数依赖集

5.3.2.1 闭包​

若 AAA 能在依赖关系 FFF 下推导出 BBB，则 BBB 是 AAA 关于 FFF 的闭包，记作：(A)F+=B(A)_{F}^{+}=B(A)F+​=B，

计算方法如下，若求 (U)F+(U)_{F}^{+}(U)F+​，其中 U=ABU=ABU=AB：

  1. 令 X0=ABX_0=ABX0​=AB。
  2. 在依赖关系 FFF 中寻找左边是 AAA、BBB 或 ABABAB 的关系，把右边并起来，令为 X1X_1X1​。
  3. 若 X1=UX_1=UX1​=U，或 X1=X0X_1=X_0X1​=X0​，则 (U)F+=X1(U)_{F}^{+}=X_1(U)F+​=X1​；否则令 X0=X1X_0=X_1X0​=X1​，重复步骤 2。

例题：

  1. 已知关系 R(A,B,C,D,E,G)R(A,B,C,D,E,G)R(A,B,C,D,E,G) 满足函数依赖关系：

    F={AB→C, D→EG, C→A, BE→C, BC→D, CG→BD, ACD→B, CE→AG}F=\{AB \rightarrow C,\ D \rightarrow EG,\ C \rightarrow A,\ BE \rightarrow C,\ BC \rightarrow D,\ CG \rightarrow BD,\ ACD \rightarrow B,\ CE \rightarrow AG\}F={AB→C, D→EG, C→A, BE→C, BC→D, CG→BD, ACD→B, CE→AG}

    求 (BD)F+(BD)^{+}_{F}(BD)F+​。

    参考答案

    解：令 (BD)0=BD(BD)^0=BD(BD)0=BD，则

    (BD)1=BD∪EG=BDEG(BD)^1=BD \cup EG=BDEG(BD)1=BD∪EG=BDEG,

    (BD)1=BDEG∪C=BDEGC(BD)^1=BDEG \cup C=BDEGC(BD)1=BDEG∪C=BDEGC,

    (BD)2=BDEGC∪AG=BDEGCA=R(BD)^2=BDEGC \cup AG=BDEGCA=R(BD)2=BDEGC∪AG=BDEGCA=R,

    于是：(BD)F+=BDEGCA(BD)^{+}_{F}=BDEGCA(BD)F+​=BDEGCA

  2. 已知关系 R(A,B,C,G,H,I)R(A,B,C,G,H,I)R(A,B,C,G,H,I) 满足函数依赖关系：

    F={A→B, A→C, CG→H, CG→I, B→H}F=\{A \rightarrow B,\ A \rightarrow C,\ CG \rightarrow H,\ CG \rightarrow I,\ B \rightarrow H\}F={A→B, A→C, CG→H, CG→I, B→H}

    求 (AG)F+(AG)^{+}_{F}(AG)F+​。

    参考答案

    解：令 (AG)0=AG(AG)^0=AG(AG)0=AG，则

    (AG)1=AG∪BC=AGBC(AG)^1=AG \cup BC=AGBC(AG)1=AG∪BC=AGBC,

    (AG)1=AGBC∪HI=AGBCHI=R(AG)^1=AGBC \cup HI=AGBCHI=R(AG)1=AGBC∪HI=AGBCHI=R,

    于是：(AG)F+=AGBCHI(AG)^{+}_{F}=AGBCHI(AG)F+​=AGBCHI

5.3.2.2 最小函数依赖集​

定义：

  • 任一函数的右边只有一个属性；
  • 不存在冗余（多余）的函数；
  • 函数的左边不能有冗余属性。

计算步骤如下，若求函数依赖 FFF 的最小依赖集 F′F'F′：

  1. 用分解规则将所有依赖右边均分解为单属性依赖。
  2. 去掉 FFF 中多余的依赖。依次尝试去掉一个依赖关系：
    1. 若删除会导致依赖关系少一个甚至多个元素，则不能去掉，否则继续尝试去掉。
    2. 假设去掉的是 X→YX \rightarrow YX→Y，在剩下的依赖关系中求 XF+X^{+}_{F}XF+​ 是否含 YYY，若包含则去掉，且下一次计算时使用去掉之后的函数依赖，否则不能去掉。
  3. 去掉各依赖左边多余的属性。依次检查左边不是单属性的依赖，假设是 XY→ZXY \rightarrow ZXY→Z，求 X→ZX \rightarrow ZX→Z 是否包含 ZZZ，若包含则去掉 YYY，即保留 X→ZX \rightarrow ZX→Z。

例题：

  1. 求函数依赖关系 FFF 的最小函数依赖集：

    F={AB→C, A→BC, B→A, C→AB, A→D}F=\{AB \rightarrow C,\ A \rightarrow BC,\ B \rightarrow A,\ C \rightarrow AB,\ A \rightarrow D\}F={AB→C, A→BC, B→A, C→AB, A→D}
    参考答案

    解：展开所有右部不为单属性的依赖：

    F1′={AB→C, A→B, A→C, B→A, C→A, C→B, A→D}F'_{1}=\{AB \rightarrow C,\ A \rightarrow B,\ A \rightarrow C,\ B \rightarrow A,\ C \rightarrow A,\ C \rightarrow B,\ A \rightarrow D\}F1′​={AB→C, A→B, A→C, B→A, C→A, C→B, A→D}

    依次尝试去掉依赖关系：

    1. 尝试去掉 AB→CAB \rightarrow CAB→C，令 (AB)0=AB(AB)^0=AB(AB)0=AB，则：

      (AB)1=AB∪CD=ABCD=R(AB)^1=AB \cup CD=ABCD=R(AB)1=AB∪CD=ABCD=R

      得 (AB)F+=ABCD(AB)^{+}_{F}=ABCD(AB)F+​=ABCD，包含 CCC，可以去掉。

    2. 尝试去掉 A→BA \rightarrow BA→B，令 (A)0=A(A)^0=A(A)0=A，则：

      (A)1=A∪CD=ACD(A)^1=A \cup CD=ACD(A)1=A∪CD=ACD，

      (A)2=ACD∪B=ACDB=R(A)^2=ACD \cup B=ACDB=R(A)2=ACD∪B=ACDB=R，

      得 (A)F+=ACDB(A)^{+}_{F}=ACDB(A)F+​=ACDB，包含 BBB，可以去掉。

    3. 尝试去掉 A→CA \rightarrow CA→C，令 (A)0=A(A)^0=A(A)0=A，则：

      (A)1=A∪D=AD(A)^1=A \cup D=AD(A)1=A∪D=AD，

      (A)2=AD=(A)1(A)^2=AD=(A)^1(A)2=AD=(A)1，

      得 (A)F+=AD(A)^{+}_{F}=AD(A)F+​=AD，不包含 BBB，不能去掉。

    4. 尝试去掉 B→AB \rightarrow AB→A，令 (B)0=B(B)^0=B(B)0=B，则：

      (B)1=B=(B)0(B)^1=B=(B)^0(B)1=B=(B)0，

      得 (B)F+=B(B)^{+}_{F}=B(B)F+​=B，不包含 AAA，不能去掉。

    5. 尝试去掉 C→AC \rightarrow AC→A，令 (C)0=C(C)^0=C(C)0=C，则：

      (C)1=C∪BD=CBD(C)^1=C \cup BD=CBD(C)1=C∪BD=CBD，

      (C)2=CBD∪A=CBDA=R(C)^2=CBD \cup A=CBDA=R(C)2=CBD∪A=CBDA=R，

      得 (C)F+=CBDA(C)^{+}_{F}=CBDA(C)F+​=CBDA，包含 AAA，可以去掉。

    6. 尝试去掉 C→BC \rightarrow BC→B，令 (C)0=C(C)^0=C(C)0=C，则：

      (C)1=C=(C)0(C)^1=C=(C)^0(C)1=C=(C)0，

      得 (C)F+=C(C)^{+}_{F}=C(C)F+​=C，不包含 AAA，不能去掉。

    7. A→DA \rightarrow DA→D 是唯一一个包含元素 DDD 的依赖，不能去掉。

    此时得到：

    F2′={A→C, B→A, C→B, A→D}F'_{2}=\{A \rightarrow C,\ B \rightarrow A,\ C \rightarrow B,\ A \rightarrow D\}F2′​={A→C, B→A, C→B, A→D}

    整理可得：

    F′={A→CD, B→A, C→B}F'=\{A \rightarrow CD,\ B \rightarrow A,\ C \rightarrow B\}F′={A→CD, B→A, C→B}
