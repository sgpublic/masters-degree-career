# 1.3 DBMS的体系结构

+ **用户与接口**：提供数据库管理员、用户和应用程序的操作接口，支持DDL、查询、更新和事务命令
+ **查询处理**：解析、优化查询并执行，实现高效的数据访问
+ **事务与并发管理**：事务开始、提交、回滚，保证并发操作的隔离性和一致性
+ **恢复与日志**：记录操作日志，支持系统崩溃后的数据恢复
+ **数据访问与缓存**：管理记录、索引和缓存，提升读写效率
+ **存储管理**：管理物理存储和数据页读写，实现数据持久化
+ **元数据与统计**：维护表结构、约束和索引等元数据，提供优化所需统计信息
