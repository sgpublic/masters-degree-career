### 2.2.2.1 列表推导式

Python 可使用以下语法生成一个列表：

```python
[<expression> for <expression> in <collection>]
```

例如：

```python
my_list = [x * x for x in range(1, 10)]
```

还可以加条件：

```python
my_list = [x * x for x in list1 if x % 2 == 0]
```

还可以同时遍历多个数组：

```python
my_list = [m + n for m in 'ABC' for n in 'XYZ']
```
